This project is a full-stack web application designed to allow users to discover, review, and rate restaurants. Built using modern web technologies, it provides a seamless and intuitive user experience. Users can browse through a list of restaurants, view details and reviews for each restaurant, and add their own reviews. The application also features sorting and filtering options to help users find restaurants based on specific criteria like price range and ratings. The backend is powered by Node.js and Express, with a PostgreSQL database, while the frontend is developed using React, ensuring a responsive and dynamic interface.
